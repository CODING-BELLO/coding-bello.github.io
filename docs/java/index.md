# Programmazione in Java

Java è un linguaggio super importante perché unisce due cose molto belle:
- rigore;
- leggibilità.

Con Java impariamo a programmare in modo più strutturato rispetto a Python, ma senza scendere subito nei dettagli più low-level di C o C++.

Programmare significa sempre la stessa cosa:
- partire da un problema;
- capire quali dati servono;
- costruire una sequenza di passi;
- trasformare quei passi in codice.

In questo percorso useremo Java per allenare:
- il ragionamento algoritmico;
- la decomposizione dei problemi;
- la progettazione di programmi ordinati;
- la programmazione a oggetti;
- i concetti base di thread e concorrenza.

## Perché Java è formativo

Java è molto usato a scuola perché:
- obbliga a essere abbastanza precisi;
- rende esplicita la struttura del programma;
- introduce bene classi, oggetti e metodi;
- permette di parlare anche di programmi più grandi e organizzati.

In pratica, Java è un ottimo ponte tra:
- programmazione di base;
- pensiero algoritmico;
- progettazione software.
