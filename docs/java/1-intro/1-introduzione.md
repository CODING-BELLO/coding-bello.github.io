# Introduzione a Java

Java è un linguaggio creato negli anni '90 da Sun Microsystems, poi passato sotto Oracle.  
È stato progettato con un'idea forte: scrivere codice portabile e affidabile.

## Una frase famosa

Una frase storica associata a Java è:

**Write once, run anywhere**

L'idea è questa: il codice Java viene compilato in **bytecode**, che può essere eseguito dalla **JVM** (Java Virtual Machine) su sistemi diversi.

## Come funziona in pratica

In forma semplificata:

- noi scriviamo un file `.java`;
- il compilatore lo trasforma in bytecode;
- la JVM esegue quel bytecode.

Quindi Java non è interpretato puro come Python, ma nemmeno compilato direttamente in eseguibile nativo come C++ nel caso classico.

## Dove si usa Java

Java è stato usato tantissimo per:
- applicazioni desktop;
- applicazioni enterprise;
- backend web;
- Android (storicamente molto importante);
- strumenti didattici per OOP.

## Perché studiarlo

Studiare Java aiuta a capire bene:
- sintassi e struttura di un programma;
- tipi di dato;
- controllo di flusso;
- metodi;
- classi e oggetti;
- eccezioni;
- thread e concorrenza.
